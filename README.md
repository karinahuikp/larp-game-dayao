# 大曜王朝 · LARP 原型（PR 1）

## 如何開啟
- 下載或 clone 本專案後，直接以瀏覽器開啟 `index.html` 即可。
- 無需額外套件或建置指令，所有資源皆為純前端檔案。

## 如何開始遊戲
1. 於封面輸入玩家名稱並按下「開始」。
2. 在角色選擇頁面點擊角色卡（六角皆可），確認後即鎖定身分。
3. 進入第一章劇本閱讀區，閱讀角色腳本並按下「進入宮門」確認鎖定。
4. 本版本僅示範第一章流程，後續章節將在後續 PR 推進。

## 檔案結構
- `index.html`：載入應用容器與主要腳本/樣式。
- `style.css`：深宮暗黑風基本樣式。
- `data.js`：遊戲資料（角色、章節、文案、任務、線索）。
- `script.js`：流程控制與畫面渲染。
