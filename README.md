# 大曜王朝 · LARP 原型（PR 2）

## 如何開啟
- 下載或 clone 本專案後，直接以瀏覽器開啟 `index.html` 即可。
- 無需額外套件或建置指令，所有資源皆為純前端檔案。

## 如何開始遊戲
1. 於封面輸入玩家名稱並按下「開始」。
2. 在角色選擇頁面點擊角色卡（六角皆可），確認後即鎖定身分。
3. 閱讀角色第一章腳本後按下「進入宮門」。
4. 進入主介面後，可於底部三頁籤切換：我的卷宗 / 本章任務 / 聚會現場。
5. 聚會現場會列出 Ch1 入座名錄；若為皇后（c5）或 GM 可推進章節示意至 Ch2。

## 檔案結構
- `index.html`：載入應用容器與主要腳本/樣式。
- `style.css`：深宮暗黑風基本樣式。
- `data.js`：遊戲資料（角色、章節、文案、任務、線索）。
- `script.js`：流程控制與畫面渲染。
